# weatherAPI

### Summary
JavaScript의 Geolocation API을 이용하여 현재 위치 정보를 받아온 후, weather API를 이용하여 오늘로부터 7일의 일기를 불러온다.

### Using API
1. Geolocation API
2. weather API - https://openweathermap.org/api
